
# Google Books App

## Kirjeldus
Rakendus kasutab Google Books API-t raamatute otsimiseks, kuvamiseks ja lemmikutesse lisamiseks.

## Funktsioonid
- Raamatute otsing
- Detailvaade
- Lemmikud (localStorage)
- React Router navigation
- Error handling + loading state

## Kasutatud API
https://www.googleapis.com/books/v1/volumes

## Käivitamine
npm install  
npm run dev
